Definición de HTML
HTML, siglas en inglés de HyperText Markup Language (‘lenguaje de marcado de hipertexto’). Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros
